Weibull Distribution

The Weibull distribution models time-to-event data — failure rates, claim durations, survival times. The shape parameter k controls whether hazard increases, decreases, or stays constant over time.

Probability Density Function — f(x; k, λ)

Density Function

f(x; k, λ) = (k / λ) · (x / λ)k−1 · e−(x/λ)k
k = shape · λ = scale · x ≥ 0
